

LaTeX stuff
--------------

The hard way
   M-x latex-mode
   or edit your .emacs
   (add-to-list 'auto-mode-alist '("\\.nw\\'" . latex-mode))

The easy way
   Use the GUI and install auctex


Block comment
--------------

highlight a region then use M-;

Markdown files
-----------------

sudo apt-get install elpa-markdown-mode

Terminal
--------------

Run a command at bottom
M-x term
